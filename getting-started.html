üìã Overview

Tippy Blender Link is a powerful Blender addon that streamlines the process of exporting 3D models as GLB files and uploading them directly to Firebase Storage and Realtime Database. This guide will walk you through the complete setup process.

What you'll be able to do:
  • Export objects or collections as optimized GLB files
  • Upload files directly to Firebase Storage
  • Store metadata in Firebase Realtime Database
  • Use presets for different quality levels (Mobile VR, PC VR, High Quality)
  • Batch export multiple objects

‚úÖ Requirements

  • Blender: Version 3.0 or higher
  • Operating System: Windows, macOS, or Linux
  • Internet Connection: Required for Firebase uploads
  • Firebase Account: Free tier is sufficient for most use cases

üî• Firebase Setup

Step 1: Create a Firebase Project

1 Go to the Firebase Console

2 Click "Create a project" or "Add project"

3 Enter your project name (e.g., "tippy-blender-uploads")

4 Choose whether to enable Google Analytics (optional)

5 Click "Create project" and wait for it to be ready

Step 2: Enable Firebase Storage

1 In the Firebase Console, click on "Storage" in the left sidebar

2 Click "Get started"

3 Choose your security rules:

// For testing (NOT for production):
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read, write: if request.auth != null;
    }
  }
}

4 Select your Cloud Storage location (choose one close to your users)

5 Click "Done"

Step 3: Enable Realtime Database

1 Click on "Realtime Database" in the left sidebar

2 Click "Create Database"

3 Select your database location

4 Choose security rules:

// For testing (NOT for production):
{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}

5 Click "Enable"

‚ö†Ô∏è Security Notice: The rules above are for testing only. For production use, implement proper security rules based on your authentication system and user permissions.

Step 4: Get Your Firebase Configuration

1 Go to Project Settings (gear icon) ‚Üí "Project settings"

2 Scroll down to "Your apps" section

3 Click "Add app" and select "Web" ( icon)

4 Register your app with a nickname (e.g., "Blender Addon")

5 Copy the configuration object:

Your Firebase Configuration

{
  "apiKey": "your-api-key-here",
  "authDomain": "your-project.firebaseapp.com",
  "databaseURL": "https://your-project.firebaseio.com",
  "projectId": "your-project-id",
  "storageBucket": "your-project.appspot.com",
  "messagingSenderId": "123456789",
  "appId": "your-app-id"
}

Save this configuration - you'll need it for the Blender addon!

Step 5: Create a Service Account (Required for Authentication)

1 In Firebase Console, go to Project Settings ‚Üí "Service accounts"

2 Click "Generate new private key"

3 Click "Generate key" in the confirmation dialog

4 A JSON file will download - keep this file secure!

üîí Security Warning: The service account key file contains sensitive credentials. Never commit it to version control or share it publicly.

‚öôÔ∏è Configuration

Configure Firebase Credentials

1 In Blender, go to Edit ‚Üí Preferences ‚Üí Add-ons

2 Find "Tippy Blender Link" and expand it

3 Click on "Preferences" to expand the settings

4 You'll see two options for Firebase configuration:

Option A: Paste Firebase Configuration (Easier)

  1. Copy your entire Firebase configuration object from the Firebase Console
  2. Paste it into the "Firebase Config JSON" text field
  3. Click "Validate Configuration"

Option B: Upload Service Account File

  1. Click the file browser icon next to "Service Account File"
  2. Select the JSON file you downloaded from Firebase
  3. Click "Load Service Account"
üí° Tip: The configuration is saved with your Blender preferences, so you only need to do this once.

Configure Export Settings

In the Tippy panel, you can choose from three export presets:

  • Mobile/VR: Optimized for Quest and mobile devices (lower poly count, compressed textures)
  • PC VR: Balanced quality for PC VR applications
  • High Quality: Maximum quality for showcases and high-end applications

üöÄ Basic Usage

Exporting a Single Object

1 Select the object or collection you want to export

2 Press N to open the sidebar (if not visible)

3 Click on the "Tippy" tab

4 Choose your export preset

5 (Optional) Enter tags for organization

6 Click "Export & Upload to Firebase"

Batch Export

1 Select multiple objects or collections

2 In the Tippy panel, click "Batch Export Selected"

3 Each object will be exported as a separate GLB file

View Upload History

The addon keeps track of your recent uploads. You can view them in the "Recent Uploads" section of the Tippy panel.

üîß Troubleshooting

Common Issues and Solutions

‚ùå "Firebase configuration not found"

Solution: Make sure you've configured Firebase credentials in the addon preferences.

‚ùå "Authentication failed"

Solution: Check that your Firebase configuration is correct and that Storage and Realtime Database are enabled.

‚ùå "File too large" error

Solution: The maximum file size is 40MB. Try using a lower quality preset or optimizing your model.

‚ùå Addon doesn't appear in Blender

Solution:

  1. Make sure you're using Blender 3.0 or higher
  2. Check that the addon is enabled in Preferences
  3. Try restarting Blender
  4. Verify the addon files are in the correct directory

‚ùå Upload fails silently

Solution: Check the Blender console (Window ‚Üí Toggle System Console on Windows) for error messages.
